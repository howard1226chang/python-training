# terminal
D:\VScode_Blue>C:\Users\Eddie\AppData\Local\Programs\Python\Python314\python.exe d:/VScode_Blue/test.py<br>
輸入:<br>
5<br>

32<br>

643<br>

234<br>

輸出:<br>
|&emsp;&emsp;5&ensp;&emsp;&ensp;32|<br>
|&emsp;643&ensp;&emsp;234|<br>
|5&emsp;&emsp;&ensp;32&emsp;&ensp;|<br>
|643&emsp;&ensp;234&emsp;|<br>
# 分析
輸出用迴圈頭尾輸出"|",需要一個函數可以補空位,宣告一個計數用於換行(每兩個換一次)
# 解
填空值的做法:print("%n.gs"%(d))
其中n = 總佔用空間,g = 取前g個字元,s = string,d = 變數名稱
在n前面可加上-號改變填空值方向
